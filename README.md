## 🎨 Features  

| Feature | Description |  
|---------|-------------|  
| 🖼️ **Artwork Gallery** | Professional display of calligraphy, miniature, illumination & painting works with full details |  
| 👩‍🎨 **Artist Profile** | Dedicated "About" section showcasing artist's background, skills & specialties |  
| 📝 **Online Ordering** | Complete order management system built with Django backend |  
| 🎯 **Specialties Showcase** | Calligraphy, Persian Illumination (Tazhib), Gol-o-Morgh, Tash'ir |  
| 📱 **Fully Responsive** | Perfect viewing experience across mobile, tablet & desktop |

| ⚡ **High Performance** | Optimized with Vite, lazy loading & modern best practices |  
| 🔒 **Secure Backend** | Django REST Framework with proper authentication & data handling |
